# Stepper-motor-closed-loop-control

## 專案簡介

這是一個使用 ESP32 實現的步進馬達閉迴路控制系統，結合 DRV8825 驅動器和 AS5600 磁性編碼器，實現精確的位置控制。

## 硬體需求

- ESP32 開發板 (ESP32-WROOM-32 或相容板)
- DRV8825 步進馬達驅動器
- NEMA 17 步進馬達 (200 步/轉)
- AS5600 磁性編碼器
- 12V 電源供應器
- 跳線和麵包板

## 接線圖

### DRV8825 連接

```
ESP32 Pin    ->  DRV8825 Pin
18           ->  STEP
19           ->  DIR
4            ->  ENABLE (建議改用GPIO4避免與I2C衝突)
16           ->  MS1
17           ->  MS2
5            ->  MS3
GND          ->  GND
12V+         ->  VMOT
12V-         ->  GND
```

### AS5600 連接 (使用 OUT 引腳模式)

```
ESP32 Pin    ->  AS5600 Pin
21           ->  OUT (類比輸出)
3.3V         ->  VCC
GND          ->  GND
```

**重要說明**:

- 此版本使用 AS5600 的 OUT 引腳進行類比讀取，不需要 I2C 連接
- OUT 引腳輸出 0-3.3V 的類比電壓，對應 0-360 度角度
- 確保 AS5600 使用 3.3V 電源供電

### 步進馬達連接

```
DRV8825      ->  步進馬達
1A           ->  A+
1B           ->  A-
2A           ->  B+
2B           ->  B-
```

## 軟體依賴

此版本使用 AS5600 的 OUT 引腳進行類比讀取，無需額外函式庫：

- 僅需 ESP32 Arduino 核心函式庫
- 不需要 AS5600 I2C 函式庫

## 功能特點

- **PID 閉迴路控制**: 實現精確的位置控制
- **微步控制**: 支援 1/16 微步，提供更平滑的運動
- **即時監控**: 串列埠顯示目前角度、目標角度和誤差
- **參數調整**: 可即時調整 PID 參數
- **指令控制**: 透過串列埠接收控制指令

## 使用方法

1. **上傳程式**: 將程式上傳到 ESP32
2. **開啟串列埠監視器**: 設定鮑率為 115200
3. **使用指令控制**:

### 控制指令

- `s<角度>` - 設定目標角度 (例如: `s90` 設定 90 度)
- `p<值>` - 設定 KP 值 (例如: `p2.5`)
- `i<值>` - 設定 KI 值 (例如: `i0.1`)
- `d<值>` - 設定 KD 值 (例如: `d0.05`)
- `e` - 啟用/停用馬達
- `r` - 重設位置為零點
- `c` - 校準零點

### 範例操作

```
s180    // 轉到180度
s-90    // 轉到-90度
p3.0    // 設定KP為3.0
i0.2    // 設定KI為0.2
d0.1    // 設定KD為0.1
e       // 停用馬達
e       // 重新啟用馬達
r       // 重設位置
c       // 校準零點
```

## PID 參數調整指南

### 初始建議值

- **KP (比例)**: 2.0 - 控制響應速度
- **KI (積分)**: 0.1 - 消除穩態誤差
- **KD (微分)**: 0.05 - 減少超調和震盪

### 調整原則

1. **KP 過大**: 系統會震盪，超調嚴重
2. **KP 過小**: 響應緩慢，無法快速到達目標
3. **KI 過大**: 系統不穩定，容易震盪
4. **KI 過小**: 無法消除穩態誤差
5. **KD 過大**: 對雜訊敏感，可能造成高頻震盪
6. **KD 過小**: 超調量大，調節時間長

## 系統特性

- **解析度**: 0.088 度 (AS5600: 12 位元，4096 步/轉)
- **微步設定**: 1/16 微步 (3200 步/轉)
- **控制頻率**: 約 1kHz
- **死區**: ±0.5 度 (避免小幅震盪)

## 故障排除

### 常見問題

1. **馬達不轉動**

   - 檢查電源連接
   - 確認 ENABLE 引腳為 LOW
   - 檢查步進馬達接線

2. **位置不準確**

   - 調整 PID 參數
   - 檢查 AS5600 磁鐵安裝
   - 確認 I2C 連接正常

3. **震盪問題**

   - 降低 KP 值
   - 增加 KD 值
   - 檢查機械部分是否有間隙

4. **編碼器讀取錯誤**
   - 檢查 I2C 接線
   - 確認 AS5600 電源為 3.3V
   - 檢查磁鐵與感測器距離(建議 1-3mm)
